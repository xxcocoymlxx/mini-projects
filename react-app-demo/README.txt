In order to install the application, please follow these steps:

1. cd to the /react-app-demo directory and run "npm install" to install node modules
2. set your PORT in src/constants.js in order to run the express server (the default port is 11088)
3. if you choose to change the port number, also change it in line 12 of the package.json
4. run "npm run dev" to start both the react app and express server
5. go on http://localhost:3000 to see the application
6. user "control+c" to quit the appilication
